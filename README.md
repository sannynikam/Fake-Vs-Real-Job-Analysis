# Fake-Vs-Real-Job-Analysis

**🔍 Project Overview**

This project focuses on analyzing job postings data to identify patterns between fake and real job listings using data cleaning, transformation, and interactive visualization techniques. The goal is to uncover insights that help in detecting fraudulent job postings and improving decision-making.


**📁 Dataset**
- Raw Data: CSV file containing unstructured job posting data
- Cleaned Data: Excel file after preprocessing and transformation
- Total Records: 3,000+ job listings


**⚙️ Workflow**

1️⃣ Data Collection
- Imported raw dataset in CSV format
- Explored structure and identified data quality issues

2️⃣ Data Cleaning (Excel)
- Removed missing and duplicate values
- Standardized formats and column names
- Filtered relevant features for analysis

3️⃣ Data Transformation (Power BI)
- Used Power Query for ETL (Extract, Transform, Load)
- Cleaned and structured data for reporting
- Created relationships and data model


4️⃣ Dashboard Development

Built interactive dashboards in Power BI
Used:
- DAX for calculated measures and KPIs
- Slicers and filters for dynamic analysis
- Visualizations (bar charts, pie charts, KPIs)

**📊 Key Insights**

- Distribution of fake vs real job postings
- Trends based on company profiles, job types, and locations
- Identification of patterns commonly found in fake job listings


**🛠️ Tools & Technologies**

- Excel – Data Cleaning
- Power BI – Data Visualization & Dashboard
- Power Query – ETL & Data Transformation
- DAX – Calculated Measures & KPIs
